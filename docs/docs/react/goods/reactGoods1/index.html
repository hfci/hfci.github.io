传递数据#

1. chldren可以通过props的方式传递#

function Demo(({ children })) {
return children
}
// 原始方式
<Demo>
<div>child</div>
</Demo>
// 通过prop传递
<Demo children={<div>child</div>}>
</Demo>
释义

两种方式是一样的，其实放在组件里面的jsx，最终就是通过props传递的

2. {}传递组件和JSX的区别#

const Test = <div></div>
function renderTest () {
return (
// 方式一
<div>{Test}</div>
)
}
function renderTest () {
return (
// 方式二
<div><Test /></div>
)
}
重要提示

  • 花括号{}内的变量，将直接变成表达式，不会被转换成JSX

    <MyComponent>
    {"My address is {{address}}."}
    </MyComponent>
    // jsx实际上会这样,而表达式不会转换
    React.createElement(MyComponent, null, "My address is {{address}}.");

  • 比如渲染lazy组件，就不能直接通过{}传递渲染

    const Text = React.lazy(() => import('text.tsx))
    function renderText () {
    return (
    <div>{Text}</div>
    )
    }
    👆就会报错

    Text并不是一个真正的组件，并不能直接渲染，需要调用React.createElement方法才行，所以有两种解决方案：

    • 手动转换
    const Text = React.lazy(() => import('text.tsx))
    function renderText () {
    return (
    <div>{React.createElement(Text)}</div>
    )
    }
    • 通过转换器转换
    const Text = React.lazy(() => import('text.tsx))
    function renderText () {
    return (
    <div><Text /></div>
    )
    }
