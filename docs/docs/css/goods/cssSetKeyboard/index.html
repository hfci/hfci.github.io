定制按钮#

前提条件

定制键盘上面的按钮的前提条件:

  1. 必须嵌套form标签
  2. 必须要有action属性
<form action="javascript:void(0)">
<input type="search" />>
</form>

有了action就会有提交动作，就会触发页面跳转。可以通过以下方式解决

  1. javascript:void(0)
<form action="javascript:void(0)"></form>
  1. 阻止默认事件prevent
<form action="." @submit.prevent>

定制键盘#

纯数字键盘#

  • type=number
<input type="number">

这种方式相当于帮助用于点击了数字键盘，还可以切回英文和中文。

  • type=tel
<input type="tel">

这种方式不能输入英文和中文，但是还是可以输入一些标点符号。

  • pattern="[0-9]*"
<input type="number" pattern="[0-9]*">

这种方式只能输入数字。

设置英文大小写#

autocapitalize它控制用户输入/编辑文本输入时文本输入是否自动大写，以及如何自动大写。属性如下：

  • off or none: 没有应用自动大写（所有字母都默认为小写字母）。
  • on or sentences: 每个句子的第一个字母默认为大写字母；所有其他字母都默认为小写字母。
  • words: 每个单词的第一个字母默认为大写字母；所有其他字母都默认为小写字母。
  • characters: 所有的字母都默认为大写。

自动填充#

填充手机号#

<input type="number" name="tel" autocomplete="tel">

warning

autocomplete和type存在冲突，比如: type=tel弹出来的键盘是不能输入+这个符号的，但是autocomplete的号码却是+86这个时候就有冲突，优先级是type最高，也就是键盘是对的，但是填充的功能失效。
