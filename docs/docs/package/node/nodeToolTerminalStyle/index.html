颜色#

nanocolors 字体颜色#

nanocolors，没有任何依赖，包体积也比较小，号称比 chalk 快几倍，能够在浏览器中使用

import { green, bold } from 'nanocolors'
console.log(
green(`Task ${bold('1')} was finished`)
)

gradient-string 字体渐变色#

gradient-string 文字渐变色

const gradient = require('gradient-string');
console.log(gradient('cyan', 'pink')('Hello world!'));

loading#

tasuku 任务运行期#

tasuku 终端最小的任务运行器，用来编排任务，显示任务 loading，pedding，finish，error 状态

import task from 'tasuku'
task('Task 1', async () => {
await someAsyncTask()
})
task('Task 2', async () => {
await someAsyncTask()
})
task('Task 3', async () => {
await someAsyncTask()
})

ora 渲染Loading#

ora 优雅的终端 loading 渲染器

import ora from 'ora';
const spinner = ora('Loading unicorns').start();
setTimeout(() => {
spinner.color = 'yellow';
spinner.text = 'Loading rainbows';
}, 1000);

listr 任务运行期#

listr 同样是用来终端编排任务，和 tasuku 主要区别在于：需要提前编排好任务

const tasks = new Listr([
{
title: 'Success',
task: () => 'Foo'
},
{
title: 'Failure',
task: () => {
throw new Error('Bar')
}
}
]);

链接#

terminal-link 创建链接#

terminal-link 终端创建可访问的链接

import terminalLink from 'terminal-link';
const link = terminalLink('My Website', 'https://sindresorhus.com');
console.log(link);

table#

cli-table3 渲染table样式#

cli-table3 终端显示表格

var Table = require('cli-table3');
// instantiate
var table = new Table({
head: ['TH 1 label', 'TH 2 label']
, colWidths: [100, 200]
});
// table is an Array, so you can `push`, `unshift`, `splice` and friends
table.push(
['First value', 'Second value']
, ['First value', 'Second value']
);
console.log(table.toString());

用户提示#

enquirer 美观的终端对话#

enquirer 时尚，美观的终端用户提示

const { prompt } = require('enquirer');
const response = await prompt({
type: 'input',
name: 'username',
message: 'What is your username?'
});
