1. portal-vue#

portal [ˈpɔːtl] 传递门；传送门

可以将slot渲染到任何位置。

import PortalVue from 'portal-vue'
Vue.use(PortalVue)
<portal to="destination">
<p>This slot content will be rendered wherever the <portal-target> with name 'destination'
is located.</p>
</portal>
<portal-target name="destination">
<-- p标签将渲染到这里 -->
</portal-target>

使用场景:

  • 可以解决一些多级嵌套出现的层级问题(比如组件内部有个弹窗，但是组件的层级比较低，可能导致弹窗不会显示在最上层)

2. vue-virtual-scroll-list#

虚拟列表，无限滚动容器

<template>
<div>
<virtual-list style="height: 360px; overflow-y: auto;" // make list scrollable
:data-key="'uid'"
:data-sources="items"
:data-component="itemComponent"
/>
</div>
</template>
<script>
import Item from './Item'
import VirtualList from 'vue-virtual-scroll-list'
export default {
name: 'root',
data () {
return {
itemComponent: Item,
items: [{uid: 'unique_1', text: 'abc'}, {uid: 'unique_2', text: 'xyz'}, ...]
}
},
components: { 'virtual-list': VirtualList }
}
</script>
