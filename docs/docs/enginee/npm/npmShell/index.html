npm update#

更新软件包。

npm update [-g] [pkg]
# 别名
aliases: up, upgrade

npm update 升级当前项目下所有的包。#

不带任何参数就是升级所有的资源包。

tip
  • npm update并不是升级npm自身的版本
  • 升级npm自身的版本可以通过再次安装的形式npm install -g npm@4.6.6

npm update -g 升级系统下全局安装的包。#

带了-g指令之后就是升级全局安装的资源包。

npm update -g whistle

npm update pkg 升级项目下面指定的包。#

npm update vue

注意事项#

  • 升级不能指定版本。
  • 升级都是基于语义化的。比如：@babel/core

如果package.json:

{
"@babel/core": "^7.7.0"
}

这种情况下会升级到7.8.4

如果package.json:

{
"@babel/core": "~7.7.0"
}

这种情况下会升级到7.7.7 (~7.7.0相当于>=7.7.0<7.8.0)
