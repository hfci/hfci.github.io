图片相关#

尽量嵌套父级#

img是一个inline类型的标签，在img加载失败的情况下，即使是设置了img的宽度和高度，也不会起作用，导致页面抖动。

<img class="img">
<style>
/* 图片加载失败，会导致宽高样式失效 */
.img {
width: 100px;
height: 100px
}
</style>
