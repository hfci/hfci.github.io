globalProperties 全局属性#

因为vue3能够挂载多个实例，所以全局属性不能直接挂载Vue实例对象上面，所以原来的方式就不在适用

  • vue3
  • vue2
// 1. 注册全局属性
const app = Vue.createApp({})
app.config.globalProperties.$http = axios
// 2. 获取实例使用
import { getCurrentInstance } from 'vue'
const app = getCurrentInstance()
app.config.globalProperties.$http
types.d.ts 添加类型提示
declare module '@vue/runtime-core' {
export interface ComponentCustomProperties {
$http: (data: object, rule: object) => boolean
}
}

Props 类型注解#

目前有两种方式提供Props类型注解：

  • PropType
  • defineProps
<script setup lang="ts">
import { defineComponent } from 'vue'
defineComponent({
props: {
name: string,
callback: {
type: Function as PropType<() => void>,
required: true
}
}
})
</script>
danger

defineProps 只能是要么使用运行时声明，要么使用类型声明, 同时使用两种声明方式会导致编译报错。

emits 自定义事件#

在vue3中，建议定义所有发出的事件，以便更好地记录组件应该如何工作。

  • setup
  • options
<script setup lang="ts">
import { defineEmits } from 'vue'
const emit = defineEmits<{
(e: 'change', id: number): void
(e: 'update', value: string): void
}>()
</script>
danger
  • defineEmits和defineProps一样，只能是要么使用运行时声明，要么使用类型声明, 同时使用两种声明方式会导致编译报错。
  • 事件通过返回true或者false以验证该事件是否有效
